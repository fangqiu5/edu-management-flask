# Flask教务管理系统
## 技术栈
Python Flask + MySQL + pymysql + Jinja2模板 + 面向对象编程
## 功能模块
1. 管理员登录权限控制（Session+装饰器校验访问）
2. 学生管理：新增、修改、删除、按学号/姓名模糊查询
3. 课程管理：课程信息CRUD、多条件检索
4. 成绩管理：成绩录入、修改、删除、按学生/课程查看成绩单、排名
5. 数据统计：学生总人数、各班/各专业人数、成绩分段分布、及格率计算
## 项目启动步骤
1. 安装依赖
pip install flask pymysql
2. 修改db_config.py中MySQL账号密码
3. 执行init_data.py自动创建数据表、生成测试数据与管理员账号
4. 运行app.py，浏览器访问 http://127.0.0.1:5000
